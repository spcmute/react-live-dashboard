React Live Dashboard

An interactive real-time dashboard integrating GitHub, Open-Meteo and CoinGecko public APIs. Dynamic charts with Recharts, custom hooks for data fetching, and responsive layout with live filter controls.

Panels

Three live data sources

Each panel connects to a different public API and updates independently based on the selected refresh interval.

GitHub Panel

Displays any public user's profile — avatar, bio, follower count — and a bar chart of star counts across their 10 most recently updated repositories. Each repo links to GitHub and shows language, stars, forks, and open issues.

Endpoint api.github.com/users/{"{user}"}/repos
Bar Chart Repo list Profile stats

Weather Panel

Shows current conditions for any city — temperature, feels-like, humidity and wind speed. Uses Open-Meteo's free geocoding to resolve city names to coordinates, then fetches live weather. No API key needed.

Endpoint api.open-meteo.com/v1/forecast
Temperature Humidity Wind speed

Crypto Panel

Live prices with 7-day sparkline charts, 24h percentage change indicators and market cap. Tracks up to 6 coins simultaneously. Data auto-refreshes at the user-selected interval using React Query's refetchInterval.

Endpoint api.coingecko.com/api/v3/coins/markets
Sparklines 24h change Market cap
Features

Built for interactivity

Dynamic Filters

Change the GitHub user, weather city, active coins, and refresh interval from the sidebar — all without reloading the page.

Auto-Refresh

Crypto data polls at 10s, 30s, 1m or 5m. GitHub and weather are cached — 1 min and 5 min respectively — to avoid unnecessary requests.

Loading Skeletons

Every panel shows shimmer placeholders while data loads, preventing layout shifts and blank states.

Recharts Integration

Bar chart for GitHub stars and sparkline line charts per crypto, both built with Recharts and fully responsive via ResponsiveContainer.

Responsive Design

At ≤640px the filter sidebar becomes a slide-in drawer. Panels stack vertically. Sparklines and market cap columns are hidden on small screens.

Error Handling

Each panel catches API errors independently and renders a descriptive error state — one failing API never breaks the rest of the dashboard.

Architecture

How it's built

Custom Hooks

useGitHub(user) Fetches profile + repos via React Query. Caches for 60s.
useWeather(city) Geocodes city name → fetches current conditions. Caches for 5m.
useCrypto(coins, interval) Polls CoinGecko at configurable interval with sparkline data.
useFilters() Global filter state with stable useCallback setters.
